Detailed Description

Declaraciones globales para el sistema de estimación orbital.

Este archivo contiene definiciones y declaraciones de variables y funciones globales, como los parámetros auxiliares, coeficientes gravitacionales y datos EOP. Estas estructuras son compartidas por todo el sistema para garantizar coherencia en los cálculos astronómicos y de dinámica orbital.
