Calcula la matriz de transformación Local Topocentric Coordinate (LTC).

Esta matriz transforma vectores desde el sistema de coordenadas topocéntrico local (Este, Norte, Cenit) al sistema de coordenadas geocéntrico (ECI).

Parameters
