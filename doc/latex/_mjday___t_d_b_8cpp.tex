\doxysubsection{Detailed Description}
Implementación de la conversión de tiempo TT a TDB (Tiempo Dinámico Baricéntrico). 

Esta función aplica una serie de correcciones periódicas para transformar el tiempo en formato TT (Tiempo Terrestre) al tiempo TDB (Tiempo Dinámico Baricéntrico), de acuerdo con modelos astronómicos estandarizados.
