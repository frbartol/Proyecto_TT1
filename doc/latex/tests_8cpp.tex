\doxysubsection{Detailed Description}
Batería de tests unitarios del sistema de dinámica orbital. 

Este archivo ejecuta pruebas unitarias sobre la clase \doxylink{class_matrix}{Matrix} y diversas funciones científicas relacionadas con la dinámica orbital, interpolación, transformaciones, aceleraciones y filtrado mediante Kalman. Los tests están organizados por módulos.

Antes de ejecutar los tests, se cargan datos reales como EOP, coeficientes GGM03S y DE430.
