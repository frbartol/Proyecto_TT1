\doxysubsection{Detailed Description}
Estructura que contiene parámetros auxiliares para los modelos físicos y astronómicos. 

Esta estructura centraliza los datos necesarios para realizar cálculos relacionados con la dinámica orbital, incluyendo fechas en MJD y activación de perturbaciones.
